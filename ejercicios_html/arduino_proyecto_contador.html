Proyecto Arduino

Este proyecto consiste en crear un contador que reacciona al pulsar un botón en base a lo aprendido. Ahora voy a decir paso a paso como lo hice.

Hardware

Un botón

Lo conectaremos a un pin de la placa, a una resistencia y al ground.

Un LED

Lo conectaremos a un pin de la placa, a una resistencia y al ground.

Proyecto final

Finalmente usaremos el visualizador de 7 segmentos para mostrar el número que se va incrementando. Para ello se conectará a dos resistencias y al ground.

Software

Botón

Este código hace que cada vez que se pulse el botón, nos lo indique hasta llegar a 9, que se reinicia.

LED

El código manda una orden al led que al pulsar el botón se encienda.

Proyecto final

El código manda una orden de un número que al pulsarse se va incrementando de 0 a 9, y cuando llega a 9 se reinicia al igual que en la consola del ordenador.
