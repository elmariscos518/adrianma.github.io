Tipos de selectores que entran en examen

Clase (class=””)

Las class permiten a CSS y Javascript seleccionar y acceder a elementos específicos a través de los selectores de class.

Identificador (id=””)

Los selectores de id de CSS buscan un elemento basado en el contenido del atributo id. El atributo id del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.

Todos los elementos (*)

Los selectores definen sobre qué elementos se aplicará un conjunto de reglas CSS.

Varios elementos (p, em, h1, h2)

Selecciona todos los elementos de la página cuya etiqueta HTML coincide con el valor del selector.

    Elementos contenidos en otros elementos (ol li)

  • Selecciona todos los elementos li que están dentro de otros elementos ol.
