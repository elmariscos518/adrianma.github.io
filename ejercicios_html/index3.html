TRABAJO DE INVESTIGACIÓN SOBRE LA WEB Y EL INTERNET

Abstract

La asignatura de Tecnologías de la Información y la Comunicación de segundo de bachillerato contempla el desarrollo de páginas web usando diferentes lenguajes de etiquetas. Por ello, resulta interesante para nosotros adquirir conocimientos previos del contexto histórico y tecnológico de la actual web. En este trabajo, mediante la investigación de diversas fuentes de la WWW (World Wide Web), profundizaremos en los aspectos referentes a la web (web 1.0, web 2.0 y sucesivas), como: sus características, historia, uso, evolución, diferencias entre unas y otras... Por último, desarrollaremos las aplicaciones de la Web 2.0, y sobre todo nos centraremos en una de ellas: Instagram, donde se contará: historia, características, uso, ventajas, desventajas, impacto... Con este trabajo se establece un marco contextual para la asignatura anteriormente mencionada, y se condensa la información encontrada en un documento de consulta para los alumnos de este mismo nivel educativo.
